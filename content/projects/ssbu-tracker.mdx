An application designed to help players of Super Smash Bros. Ultimate track their gameplay statistics, monitor progress, and analyze performance over time.

###### Features

- **Match Recording**: Log details of each match played.
- **Statistics Analysis**: Provides insights into win rates, character usage, and more.
- **Progress Tracking**: Visualizes improvement trends over time.
